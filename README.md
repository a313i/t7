# FutureX

موقع تعريفي احترافي بتصميم داكن (Dark Theme) متوافق مع جميع الأجهزة، جاهز للنشر على GitHub Pages.

## المميزات

- تصميم احترافي
- دعم اللغة العربية (RTL)
- متوافق مع الهواتف والأجهزة اللوحية
- سريع وخفيف
- HTML5
- CSS3
- JavaScript
- تأثيرات وحركات سلسة
- بطاقات Glassmorphism
- جاهز للتعديل

## طريقة التشغيل

1. قم بتنزيل المشروع.
2. افتح ملف `index.html` في المتصفح.

أو

ارفع المشروع إلى GitHub.

ثم قم بتفعيل GitHub Pages من:

Settings → Pages

واختر:

```
Deploy from a branch
```

ثم:

```
main
```

ثم:

```
/(root)
```

بعدها سيتم إنشاء رابط الموقع تلقائياً.
